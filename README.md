# Living Library

Premium Vite + React module for a calm, editorial "Living Library" experience.

## Dev sanity

```bash
npm install
npm run dev
npm run build
```

## Routes

- `/` — Home landing page
- `/library` — Searchable resource library
- `/transmissions` — Transmission archive
- `/transmissions/:slug` — Transmission detail reading page
- `/about` — Project overview

## Content sources

- `src/data/resources.json`
- `src/content/transmissions/*.md`
